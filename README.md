# Stock Market Data Analysis Project

## Overview
This project was completed during my Data Analysis internship at Codveda Technologies.

The project focuses on analyzing historical stock market data, building predictive models using Python, and creating an interactive Power BI dashboard to visualize insights.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Power BI

## Project Components

### Python Analysis
The Python notebook includes:
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Regression and classification modeling
- Data visualization

### Power BI Dashboard
The Power BI dashboard presents key insights including:
- Stock price trends
- Trading volume analysis
- Price comparisons across different stocks
- Interactive filters for deeper analysis
